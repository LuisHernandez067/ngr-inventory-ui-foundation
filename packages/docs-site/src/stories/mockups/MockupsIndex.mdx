# Mockups — Pantallas navegables

Las pantallas en esta sección son mockups funcionales construidos con Bootstrap 5, ui-core y ui-patterns. Los datos son simulados con MSW v2.

## Mockups disponibles

| Sección           | Pantallas                                        |
| ----------------- | ------------------------------------------------ |
| **Autenticación** | Login · Páginas de error (403, 404)              |
| **Dashboard**     | Panel principal con KPIs y alertas               |
| **Productos**     | Lista de productos · Detalle/edición de producto |

## Cómo funcionan

Cada mockup usa handlers MSW para simular las respuestas de la API. Podés cambiar el escenario usando el toolbar de temas o modificando el parámetro `msw` de la historia.

> Estas pantallas son fuente de referencia para el diseño del prototipo real, no código de producción.
