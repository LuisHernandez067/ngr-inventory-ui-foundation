# Sistema de diseño NGR Inventory

Bienvenido al portal de documentación viva de NGR Inventory. Este Storybook es la fuente de verdad para componentes, patrones, tokens de diseño y mockups del sistema.

## Capas del sistema

| Capa            | Descripción                                                          |
| --------------- | -------------------------------------------------------------------- |
| **Tokens**      | Variables de diseño: colores, espaciado, tipografía, bordes, sombras |
| **UI Core**     | Componentes base: Button, Badge, Alert, Spinner, etc.                |
| **UI Patterns** | Patrones operativos: DataTable, Pagination, SearchBar, etc.          |
| **Mockups**     | Pantallas navegables con datos simulados vía MSW                     |

## Cómo usar este portal

- Navegá el sidebar para explorar por capa
- Usá el toolbar superior para cambiar el tema (claro, oscuro, cálido, frío)
- Las historias con escenarios MSW muestran estados: carga, vacío, error

## Stack técnico

Bootstrap 5 · TypeScript · Vite · Storybook 8 · MSW v2
