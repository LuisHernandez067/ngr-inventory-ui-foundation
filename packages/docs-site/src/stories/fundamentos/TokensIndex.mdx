# Tokens de diseño

Los tokens son las variables de diseño del sistema. Toda decisión visual — color, espacio, tipografía, borde — está definida aquí.

## Grupos de tokens

- **Colores** → paleta NGR Blue, semánticos, neutrales Slate, comparación por tema
- **Espaciado** → escala modular de 4px a 96px
- **Tipografía** → familias, escala de tamaños, pesos
- **Bordes** → radio de esquinas y sombras

> Ningún valor de color, espaciado o tipografía debe ser hardcodeado. Siempre usá los tokens.
